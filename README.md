Run the App Locally
---------------------
Create a virtual environment:
* On macOs and Linux:
````
python3 -m venv venv
````
* On Window:
````
venv\Scripts\activate
````
Install the dependenvies:
* On macOs and Linux:
````
pip3 install -r requirements.txt
````
* On Windows:
````
pip install -r requirements.txt
````
Frontend
````
npm create vite@latest .
````
